# Welcome to React-Native Expo app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

## Supabase changes

Row‐Level Security (RLS) set to disabled in order to easily access database table entries

## TanStack Query Setup

QueryClientProvider and queryClient is imported, and wrapped as top-level wrapper in my-app>app so childs components can access the useQuery
